# Usb Format

GUI Application to format USB Devices in Ubuntu


## Installation

For Python:
```bash
sudo apt-get update sudo apt-get install python3.
```

Install pip
```bash
sudo apt install python3-pip
```
Install tkinter
```bash
 sudo apt-get install python3-pil.imagetk
```

## Recommended

Use Conda Virtual Environment
